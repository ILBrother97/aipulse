# AIPulse - Your Personal AI Tools Hub

A modern, intuitive platform to organize, launch, and track all your AI tools in one place. Built with React 19, TypeScript, and TailwindCSS for a seamless user experience.

## ✨ Features

### Core Functionality
- **Tool Management**: Discover, organize, and launch AI tools quickly
- **Smart Collections**: Create custom collections to categorize your tools
- **Usage Analytics**: Track tool launches, usage patterns, and trends over time
- **Quick Access**: Favorites and recently used tools for instant access
- **Search & Filter**: Find tools by category, name, or usage frequency

### AI-Powered Assistant
- Intelligent chatbot to help you discover tools
- Context-aware conversations that remember previous interactions
- Rich formatting with markdown support
- Actionable responses with quick navigation buttons
- Personalized recommendations based on your usage patterns

### User Experience
- **Dark/Light Mode**: Full theme support with smooth transitions
- **Responsive Design**: Optimized for desktop, tablet, and mobile
- **Modern UI**: Clean interface with smooth animations
- **Local Storage**: Data persists in your browser

## 🛠 Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: TailwindCSS
- **Animations**: Framer Motion
- **State Management**: Zustand with persistence
- **Icons**: Lucide React
- **Markdown**: react-markdown

## � Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

##  Project Structure

- `/src/components` - React components (features, layout, modals, ui)
- `/src/stores` - Zustand state management
- `/src/utils` - Utility functions and helpers
- `/src/types` - TypeScript type definitions
- `/public` - Static assets

## � Privacy

All data is stored locally in your browser. No account required for basic functionality.
